# SOCxProjects

SOC Analyst Home Lab Projects - A collection of practical SOC analyst labs and exercises designed to build real-world security operations skills.

## About

This repository contains a series of hands-on home lab exercises that simulate real SOC analyst scenarios. Each lab is designed to progressively build your skills in threat detection, incident response, SIEM operations, security tooling, and automation.

## Labs

| Lab | Title | Skills Covered |
|-----|-------|----------------|
| [Lab 1](./Lab1-CSIRT-Establishment) | CSIRT Establishment | CSIRT frameworks, incident handling processes, playbooks |
| [Lab 2](./Lab2-Phishing-Investigation) | Phishing Investigation | Email header analysis, IOC extraction, threat intelligence |
| [Lab 3](./Lab3-SIEM-Build) | SIEM Build & Configuration | SIEM deployment, log ingestion, parsing, dashboards |
| [Lab 4](./Lab4-Splunk-SOC) | Splunk SOC Operations | SPL queries, correlation searches, alerting, dashboards |
| [Lab 5](./Lab5-Python-SOC-Tool) | Python SOC Tool Development | Python scripting, API integration, automation |
| [Lab 6](./Lab6-Build-A-Home-Lab) | Build a Home Lab | Virtualization, networking, lab architecture |

## Skills Developed

- Threat detection and monitoring
- Incident response and handling
- Log analysis and correlation
- SIEM operations (Splunk, ELK, Wazuh)
- Security automation and scripting
- Home lab architecture and design
